Tujuan Proyek 
● Membangun aplikasi chat berbasis web yang dapat diakses pengguna untuk 
    berinteraksi dengan model OpenAI. 
● Menyimpan log percakapan dalam basis data MySQL untuk analisis lebih lanjut. 
● Memanfaatkan AWS Cloud untuk memastikan skalabilitas, ketersediaan, dan 
    keamanan aplikasi.
